{\footnotesize\ttfamily typedef \mbox{\hyperlink{vector_8h_a1ec1d8ee2a4766c9427ebf2a1c221f52}{Vector}} \mbox{\hyperlink{stack_8h_abf32c4253c172c0ef132cc56631a7b74}{Stack}}}



An STL-\/like Stack. 

This is a vector based stack, meaning all operations on it are in amortized O(1). It supports the following operations\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{stack_8h_a91973823ceed4a4168119c6dfcb21c58}{Stack\+Push}} Adds a new element to the top of the stack
\item \mbox{\hyperlink{stack_8h_ad00690201a5f1a2efe7c835f3576cff3}{Stack\+Pop}} Removes an item from the top of the stack and returns it
\item \mbox{\hyperlink{stack_8h_a05998a4459363b0a1ecc32c85b59acdc}{Stack\+Peek}} Returns the item at the top of the stack
\end{DoxyItemize}

It can be created and deallocated using \mbox{\hyperlink{stack_8h_a8b28dfac0e0018a0a0bacd2dfec85fab}{New\+Stack}} and \mbox{\hyperlink{stack_8h_aee7c29baf6effa1d9e0d9f86c9f25050}{Delete\+Stack}} respectively. To check its size and test if it is empty, use \mbox{\hyperlink{stack_8h_a15de68e422c2a8a50421e4244a54d758}{Stack\+Size}} and \mbox{\hyperlink{stack_8h_ac536c649ee974409593cfd5bab184635}{Stack\+Empty}}. It can be optimized for size by shrinking it to the number of elements it has using \mbox{\hyperlink{stack_8h_a23c5715a8dbd80c7c1288e18021b6922}{Stack\+Optimize}}; however, this operation is in O(n).

\begin{DoxyNote}{Note}
Even though this is only a typedef, do not use \mbox{\hyperlink{vector_8h_a1ec1d8ee2a4766c9427ebf2a1c221f52}{Vector}} as stack for improved code clarity 
\end{DoxyNote}
