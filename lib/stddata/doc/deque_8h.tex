{\footnotesize\ttfamily typedef \mbox{\hyperlink{list_8h_a29731fb23c4f8fd187f12b1d78536c30}{List}} \mbox{\hyperlink{deque_8h_aebcafdfec252d8be3c2c921866b6c720}{Deque}}}



An STL-\/like Deque. 

This is a linked-\/list-\/based deque, meaning all operations on it are in O(1). It supports the following operations\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{deque_8h_a1c4e6f3178a1f0d3b5b1a89da2576890}{Deque\+Push\+Front}} Adds a new element to the front of the deque
\item \mbox{\hyperlink{deque_8h_a75da2404e71b362f765e23a631a88051}{Deque\+Push\+Back}} Adds a new element to the back of the the deque
\item \mbox{\hyperlink{deque_8h_a14bd19b4d3b39fc78822947a0af71749}{Deque\+Pop\+Front}} Removes an item from the front of the deque and returns it
\item \mbox{\hyperlink{deque_8h_a450460284f6c3e008bc15673ff5b5f37}{Deque\+Pop\+Back}} Removes an item from the back of the deque and returns it
\item \mbox{\hyperlink{deque_8h_a0f4fbbc852a41900a3147f9b293f564a}{Deque\+Peek\+Front}} Returns the item at the front of the deque
\item \mbox{\hyperlink{deque_8h_a6759eff335d14d711bbd89fef037bc61}{Deque\+Peek\+Back}} Returns the item at the back of the deque
\end{DoxyItemize}

It can be created and deallocated using \mbox{\hyperlink{deque_8h_a4aba2fc43a7fe818348bca7654d9c27a}{New\+Deque}} and \mbox{\hyperlink{deque_8h_ab8595bdfd4cec494c0e9cdb61ad1d8d4}{Delete\+Deque}} respectively. To check its size and test if it is empty, use \mbox{\hyperlink{deque_8h_a136271a75ea6c1c77e0034781c6d0af6}{Deque\+Size}} and \mbox{\hyperlink{deque_8h_a93acb04d47d4dbf9f44ff6dad6cf4716}{Deque\+Empty}}

\begin{DoxyNote}{Note}
Even though this is only a typedef, do not use \mbox{\hyperlink{list_8h_a29731fb23c4f8fd187f12b1d78536c30}{List}} as deque for improved code clarity 
\end{DoxyNote}
