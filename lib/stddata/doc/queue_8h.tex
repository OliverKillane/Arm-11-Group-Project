{\footnotesize\ttfamily typedef \mbox{\hyperlink{deque_8h_aebcafdfec252d8be3c2c921866b6c720}{Deque}} \mbox{\hyperlink{queue_8h_aa02f3c475f67682e7fee6b14daafc7fe}{Queue}}}



An STL-\/like Queue. 

This is a linked-\/list-\/based queue, meaning all operations on it are in O(1). It supports the following operations\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{queue_8h_af12217f5b87b0ca1ce5898bc11d15a83}{Queue\+Push}} Adds a new element to the back of the the queue
\item \mbox{\hyperlink{queue_8h_aa99c0fe94271fb348afc2f45dc9b0640}{Queue\+Pop}} Removes an item from the front of the queue and returns it
\item \mbox{\hyperlink{queue_8h_a0f0cf1cb69b15b11b32f3f12623722a8}{Queue\+Peek\+Front}} Returns the item at the front of the queue
\item \mbox{\hyperlink{queue_8h_aca43aa9974105064889b4d3446b2f657}{Queue\+Peek\+Back}} Returns the item at the back of the queue
\end{DoxyItemize}

It can be created and deallocated using \mbox{\hyperlink{queue_8h_a6f3600b4ee350dd40e542111df4d11dd}{New\+Queue}} and \mbox{\hyperlink{queue_8h_a1e0f7956bca711e20922fe187752ac38}{Delete\+Queue}} respectively. To check its size and test if it is empty, use \mbox{\hyperlink{queue_8h_a50ab3e4ec9fb965a8de69b54d277ef5f}{Queue\+Size}} and \mbox{\hyperlink{queue_8h_a1d800d56243070bc9a72733cd3598561}{Queue\+Empty}}

\begin{DoxyNote}{Note}
Even though this is only a typedef, do not use \mbox{\hyperlink{list_8h_a29731fb23c4f8fd187f12b1d78536c30}{List}} or \mbox{\hyperlink{deque_8h_aebcafdfec252d8be3c2c921866b6c720}{Deque}} as a queue for improved code clarity 
\end{DoxyNote}
