A decision tree. 

This is a decision tree, which works like a map for sequences of items\+: It supports the following operations\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{decision__tree_8h_a81605c38d1d32df2f6b21a2d7b7f1b61}{Decision\+Tree\+Insert}} Adds a sequence of items and the corresponding value
\item \mbox{\hyperlink{decision__tree_8h_a516766eddb32d1c8653b66df43248ee7}{Decision\+Tree\+Query}} Retreives the value of a sequence in the decision tree
\end{DoxyItemize}

It can be created using \mbox{\hyperlink{decision__tree_8h_a5e59d503dfd7b2e391b1b844c6d7ac6f}{New\+Empty\+Decision\+Tree}} and deallocated using \mbox{\hyperlink{decision__tree_8h_a9f099acb42f72734247adddd8b2db707}{Delete\+Decision\+Tree}} To check its size and test if it is empty, use \mbox{\hyperlink{decision__tree_8h_ab1f2d62ca1b8b95c952a1e94639e56ce}{Decision\+Tree\+Size}} and \mbox{\hyperlink{decision__tree_8h_aff95812250b0fe6430fc26ffe58025a7}{Decision\+Tree\+Empty}}

\begin{DoxyNote}{Note}
Even though this is only a typedef, do not delcare it as a pointer to \mbox{\hyperlink{structDecisionTreeRepr}{Decision\+Tree\+Repr}} for improved code clarity 
\end{DoxyNote}
