{\footnotesize\ttfamily typedef \mbox{\hyperlink{structVectorRepr}{Vector\+Repr}}$\ast$ \mbox{\hyperlink{vector_8h_a1ec1d8ee2a4766c9427ebf2a1c221f52}{Vector}}}



An STL-\/like Vector. 

This is a vector implemented using allocated size doubling, meaning that all of the basic operations on it are in O(1) (except for \mbox{\hyperlink{vector_8h_a75cb33aaeaa6c056db23afc8a9333c0e}{Vector\+Forced\+Resize}}, \mbox{\hyperlink{vector_8h_a79cef7b5ee3ba33ae24879756b4448f3}{Vector\+Forced\+Clear}} and \mbox{\hyperlink{vector_8h_a113c71d0357a7c9c41138d47ca8a0dfc}{Vector\+Resize}} which are all pessimistically linear). It supports the following operations\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{vector_8h_a03c6b45e90461152283eb0c0d881e0af}{Vector\+Set}} Sets a value of the vector at a specific index (in linear time complexity)
\item \mbox{\hyperlink{vector_8h_a14e2387ae1fdd3002aafa82de0309954}{Vector\+Get}} Returns a value of the vector at a specific index (in linear time complexity)
\item \mbox{\hyperlink{vector_8h_a113c71d0357a7c9c41138d47ca8a0dfc}{Vector\+Resize}} Resizes a vector
\item \mbox{\hyperlink{vector_8h_aba41a249474323bb42de377f639e3546}{Vector\+Clear}} Resizes a vector to 0
\item \mbox{\hyperlink{vector_8h_a75cb33aaeaa6c056db23afc8a9333c0e}{Vector\+Forced\+Resize}} Resizes a vector and reallocates it so that the allocated size is as close to the actual size as possible (no smaller than \mbox{\hyperlink{vector_8h_a497bdb031760806802fb94ec019c22a4}{MIN\+\_\+\+VECTOR\+\_\+\+SIZE}} )
\item \mbox{\hyperlink{vector_8h_a79cef7b5ee3ba33ae24879756b4448f3}{Vector\+Forced\+Clear}} Clears a vector and reallocates it to an allocated size of \mbox{\hyperlink{vector_8h_a497bdb031760806802fb94ec019c22a4}{MIN\+\_\+\+VECTOR\+\_\+\+SIZE}}
\item \mbox{\hyperlink{vector_8h_a8d24971c5e38a2f8a72fa708f7b733a2}{Vector\+Push\+Back}} Adds an element at the end of the vector
\item \mbox{\hyperlink{vector_8h_a0f81565d2fb270bb672b2a12aef6dc1d}{Vector\+Pop\+Back}} Removes an element from the end of the vector
\end{DoxyItemize}

It can be created using \mbox{\hyperlink{vector_8h_a5f7dcdbcb6ee8104b7f87aa956737672}{New\+Empty\+Vector}}, \mbox{\hyperlink{vector_8h_a4e0a14677f161ddd1f0d528201cd9a49}{New\+Vector}} or \mbox{\hyperlink{vector_8h_ac6f95463b386efcad9a84297171be63a}{New\+Filled\+Vector}}, and deallocated using \mbox{\hyperlink{vector_8h_a3b82fc0e73ef5a5df2397bfbd714d07c}{Delete\+Vector}}. To check its size and test if it is empty, use \mbox{\hyperlink{vector_8h_a550413fd2710f9f8f300596c5ec7dd81}{Vector\+Size}} and \mbox{\hyperlink{vector_8h_acb0090322409a78caa9ddbfc6039f9a7}{Vector\+Empty}}

Even though this is only a typedef, do not delcare it as a pointer to \mbox{\hyperlink{structVectorRepr}{Vector\+Repr}} for improved code clarity 
\end{DoxyNote}
