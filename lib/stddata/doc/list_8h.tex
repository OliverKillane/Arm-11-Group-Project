An STL-\/like Linked List. 

This is a linked list, meaning that all item insertion operations are in O(1), but accessing items inside of the list is in O(n). It supports the following operations\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{list_8h_ae760e2adf45640233d1cf89d9327ee5e}{List\+Push\+Front}} Adds a new element to the front of the list
\item \mbox{\hyperlink{list_8h_ad8b34f2e311fd172baf716d1fc0e437d}{List\+Push\+Back}} Adds a new element to the back of the the list
\item \mbox{\hyperlink{list_8h_ab7c989cfad01ae9787828faf84a8635d}{List\+Pop\+Front}} Removes an item from the front of the list and returns it
\item \mbox{\hyperlink{list_8h_a0e21b7e0890aa1cb0cd1e550eda345a8}{List\+Pop\+Back}} Removes an item from the back of the list and returns it
\item \mbox{\hyperlink{list_8h_aa1321fa944023bf73aad868ce1415f81}{List\+Front}} Returns the item at the front of the list
\item \mbox{\hyperlink{list_8h_afa9bf84f392d7efc239644857b0618ef}{List\+Back}} Returns the item at the back of the list
\item \mbox{\hyperlink{list_8h_a72580a25032c3f873b03d37d5b678db4}{List\+Set}} Sets a value of the list at a specific index (in linear time complexity)
\item \mbox{\hyperlink{list_8h_ae4197c5aea01174e287d81e0a6ab73d7}{List\+Get}} Returns a value of the list at a specific index (in linear time complexity)
\end{DoxyItemize}

It can be created using \mbox{\hyperlink{list_8h_a46eab2bdab734fc9931d1d7fd76bea22}{New\+Empty\+List}}, \mbox{\hyperlink{list_8h_ad775bb87a4fb893b83ac9fbebbcf176e}{New\+List}} or \mbox{\hyperlink{list_8h_ab18b5d535a373ad6273d947d8e327f33}{New\+Filled\+List}}, and deallocated using \mbox{\hyperlink{list_8h_ab1b1b2f3e2d98df3c4909ebda1d213ee}{Delete\+List}}. To check its size and test if it is empty, use \mbox{\hyperlink{list_8h_ab924e88eb714daae5c4cbd469563308e}{List\+Size}} and \mbox{\hyperlink{list_8h_a2a50dce680502655d689d279c74e3cc5}{List\+Empty}}

\end{DoxyNote}
