An STL-\/like hash map. 

This is a hash map, meaning that all operations are in amortized O(1)\+: It supports the following operations\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{map_8h_a730e6feb7d729a61254f50f0629bd2f6}{Map\+Query}} Checks if a key is in the map
\item \mbox{\hyperlink{map_8h_a857961b57b460c9d24a7896eab60b0f9}{Map\+Get}} Retreives the value of a key in the map
\item \mbox{\hyperlink{map_8h_ac88f74654453435f5d9add302d2acc9b}{Map\+Set}} Sets a value of a key in the map
\item \mbox{\hyperlink{map_8h_ad1437751316b1361c495b0fe82edecf3}{Map\+Remove}} Removes a key from a map
\end{DoxyItemize}

It can be created using \mbox{\hyperlink{map_8h_a5fd51eb954f85c4e90c8f59666c5adb7}{New\+Empty\+Map}} or \mbox{\hyperlink{map_8h_a30fda3f2551d966e9b2bbda843f49f44}{New\+Filled\+Map}}, and deallocated using \mbox{\hyperlink{map_8h_a90a707f413cb9c5fcd78b4d996d2e322}{Delete\+Map}} To check its size and test if it is empty, use \mbox{\hyperlink{map_8h_a1fb3dd5b0244000d3fc5439d2cc6c916}{Map\+Size}} and \mbox{\hyperlink{map_8h_a77077961fe2d87eee166d8d70f818aa9}{Map\+Empty}}

\begin{DoxyNote}{Note}
Even though this is only a typedef, do not delcare it as a pointer to \mbox{\hyperlink{structMapRepr}{Map\+Repr}} for improved code clarity 
\end{DoxyNote}
\begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{structMapIterator}{Map\+Iterator}} to loop over a map 
\end{DoxySeeAlso}
