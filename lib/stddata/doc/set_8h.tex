An STL-\/like hash set. 

This is a hash set, meaning that all operations are in amortized O(1)\+: It supports the following operations\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{set_8h_af86fef6e8408f08f962ffc291424879f}{Set\+Query}} Checks if an item is in the set
\item \mbox{\hyperlink{set_8h_a8cecc1a093d2a78d4d8a33c769c8b48f}{Set\+Insert}} Adds an item to a set
\item \mbox{\hyperlink{set_8h_acf857db03e312884f6f21f093f76b1db}{Set\+Remove}} Removes an item from a set
\end{DoxyItemize}

It can be created using \mbox{\hyperlink{set_8h_ab4e2f9c3e0c8526f2f3c2951ca2eb87e}{New\+Empty\+Set}} or \mbox{\hyperlink{set_8h_abc3218a1be6b71dae2e3d4b0a67d0a6b}{New\+Filled\+Set}}, and deallocated using \mbox{\hyperlink{set_8h_a9df8e6838a095ebb986d65a4e0fe3fef}{Delete\+Set}} To check its size and test if it is empty, use \mbox{\hyperlink{set_8h_af53b9fce9f4e3d6daecba1839ea2c1d5}{Set\+Size}} and \mbox{\hyperlink{set_8h_a05048d28a6ef10cb46fcab72b06fb24f}{Set\+Empty}}

\begin{DoxyNote}{Note}
Even though this is only a typedef, do not delcare it as a pointer to \mbox{\hyperlink{structSetRepr}{Set\+Repr}} for improved code clarity 
\end{DoxyNote}
